## 描述

系统梳理 LLM 三十年技术主线:从 n-gram 统计预测 → Transformer 注意力机制 → GPT 上下文学习 → RLHF/GRPO 对齐 → AI Agent + Harness 工程范式。

核心命题:LLM 正完成从"生成文本"到"理解世界并执行任务"的范式跃迁,背后是三条主线交织:
- **信息聚合**:从局部 n-gram 到全局注意力,语义压缩能力的不断增强
- **能力涌现**:scaling law + 训练数据/算法纪律共同促成的能力跳变
- **驾驭工程(Harness Engineering)**:从单一 prompt 升级为 agent + 工具 + 反馈循环的完整执行装置
